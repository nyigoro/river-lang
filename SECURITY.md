# Security Policy

## Reporting a Vulnerability

If you discover a security-sensitive issue, please report it privately. Do not
open a public issue or post exploit details publicly. Contact the maintainers
via the private security reporting channel used for this repository.

## Scope

Security reports should include enough detail to reproduce the issue, affected
components, and potential impact. If possible, include a minimal proof of
concept and the version or commit hash.

## Coordinated Disclosure

We will acknowledge receipt, investigate, and coordinate disclosure timelines
with the reporter.
